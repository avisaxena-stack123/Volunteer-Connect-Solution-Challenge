# Volunteer Connect - Solution Challenge 2026
*Team Name:* Hunarbaaz  
*Team Leader:* Avi Saxena  

## Project Overview
Volunteer Connect is a prototype designed to streamline community service coordination using Google AI. 

## Key Features
- Centralized Task Dashboard
- Real-time Assignee Tracking
- AI-generated communication drafts for teammates

## Technologies Used
- Google Gemini (Flash/Pro)
- Firebase Studio
- Google Cloud Platform
